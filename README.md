Smart Resource Allocation System

Description:
This is a console-based Java application that simulates hostel room allocation for students.

Features:
- Add student and room details
- Allocate rooms based on student priority
- View, search, and delete room allocations

Technologies Used:
- Java
- Object-Oriented Programming (OOP)
- Collections (ArrayList, HashMap)

What I Learned:
Through this project, I gained a better understanding of object-oriented programming concepts and how to use data structures like ArrayList and HashMap in real scenarios.
I also learned how to design simple allocation logic and improved my problem-solving skills while working on this project.# smart-resource-allocation-system
Java project for hostel room allocation using priority logic

How to Run
1. Compile the program:
   javac AllocationApplication.java
2. Run the program:
   java AllocationApplication
3. Follow the menu options in the console
